# 🌎 LitterQuitter

LitterQuitter is a cross-platform mobile application that gamifies community litter collection by encouraging users to clean up their neighborhoods, track their environmental impact, and compete with friends through points, streaks, and achievements.

Built with React Native and Expo, LitterQuitter makes environmental action engaging, measurable, and rewarding.

---

## Features

- 📍 Record litter cleanup sessions
- 🗺️ View cleanup locations on an interactive map
- 📸 Upload photos of completed cleanups
- 🏆 Earn points, badges, and maintain streaks
- 📊 Track personal environmental impact
- 👥 Compare progress with the community through leaderboards

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/LitterQuitter.git
cd LitterQuitter
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the Expo development server

```bash
npx expo start
```

### 4. Run the app

Download **Expo Go** on your mobile device.

Then, after running:

```bash
npx expo start
```

choose one of the following:

- Press **i** to open the iOS Simulator (macOS only)
- Press **a** to open an Android emulator
- Scan the QR code with **Expo Go** to run on a physical device
